# Attendance-marking-system

A simple web-based Attendance Marking System that allows employees to check in, check out, and view their attendance history.  
This project is built using HTML, CSS, and JavaScript.

---

 Features

Employee Check-In  
Employee Check-Out  
Attendance History Table  
Current Date & Time Recording  
Simple and Clean UI  
Logout Option  

Technologies Used

- HTML5
- CSS3
- JavaScript

Project Structure
attendance-marking-system/
│── index.html
│── style.css
│── script.js
│── README.md


How It Works

1. Employee clicks **Check In**
   - The current date and time are stored.
2. Employee clicks **Check Out**
   - The check-out time is updated for the same record.
3. The attendance is displayed in the **Attendance History Table**.
